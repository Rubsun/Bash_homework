#Скрипт по конвертации текстовых файлов
##Функции
###Можно выбрать два пути нахождения фалов: конвертация фалов в нынешней папке, передать пусть до определенной папки.
###Создание папки для сохранения фалов
##Особенности
###Конвертирует все файлы нужного расширения, без возможности выбора отдельных.
